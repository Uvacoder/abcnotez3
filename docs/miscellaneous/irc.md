## IRC commands

### Register your nick

```
/msg NickServ REGISTER <password> <email>
```

### Identify your nick

```
/msg NickServ IDENTIFY <nick> <password>
```

### Change password

```
/msg NickServ SENDPASS <account_name>
/msg NickServ SET PASSWORD <new_password>
```
