## IRC commands

- Register your nick:

  ```
  /msg NickServ REGISTER YourPassword youremail@example.com
  ```

- Identify your nick:

  ```
  /msg NickServ IDENTIFY YourNick YourPassword
  ```

- Change password:

  ```
  /msg NickServ SENDPASS youraccountnamehere
  /msg NickServ SET PASSWORD newpasswordhere
  ```
